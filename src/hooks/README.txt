# Directorio de Hooks

Este directorio contiene hooks personalizados reutilizables para toda la aplicación.

## Estructura
- useFetch.ts: Hook para manejo de peticiones HTTP
- useForm.ts: Hook para manejo de formularios
- useAuth.ts: Hook para gestión de autenticación
- useTheme.ts: Hook para manejo de temas

## Propósito
El directorio hooks centraliza la lógica reutilizable de la aplicación:
- Reduce la duplicación de código
- Facilita el mantenimiento de la lógica de negocio
- Mejora la consistencia en el manejo de estado
- Permite la reutilización de lógica común entre componentes
