# Directorio de Assets

Este directorio contiene todos los recursos estáticos de la aplicación.

## Estructura
- images/: Almacena todas las imágenes utilizadas en la aplicación
- fonts/: Contiene las fuentes personalizadas
- videos/: Almacena archivos de video si son necesarios

## Propósito
El directorio assets centraliza la gestión de recursos estáticos, facilitando:
- Organización clara de recursos multimedia
- Optimización de carga de recursos
- Control de versiones de assets
- Reutilización de recursos en toda la aplicación
