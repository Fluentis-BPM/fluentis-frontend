# Directorio de Types

Este directorio contiene las definiciones de tipos TypeScript utilizadas en toda la aplicación.

## Contenido
- authTypes.ts: Tipos relacionados con autenticación y usuarios
- processTypes.ts: Tipos para la gestión de procesos

## Propósito
Las definiciones de tipos en este directorio:
- Proporcionan tipado estático para la aplicación
- Mejoran la detección de errores en tiempo de desarrollo
- Facilitan el mantenimiento del código
- Documentan la estructura de datos

## Uso
Cada archivo de tipos debe:
- Definir interfaces y tipos relacionados
- Mantener la consistencia en la nomenclatura
- Incluir documentación JSDoc cuando sea necesario
- Evitar la duplicación de tipos
