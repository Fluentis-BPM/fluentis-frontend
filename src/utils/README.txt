# Directorio de Utils

Este directorio contiene utilidades y funciones auxiliares utilizadas en toda la aplicación.

## Contenido
- api.ts: Configuración y funciones para comunicación con APIs
- helpers.ts: Funciones auxiliares generales

## Propósito
Las utilidades en este directorio:
- Centralizan funciones comunes
- Evitan duplicación de código
- Proporcionan funcionalidad reutilizable
- Facilitan el mantenimiento

## Uso
Cada utilidad debe:
- Tener un propósito específico y claro
- Estar bien documentada
- Ser fácilmente testeable
- Seguir principios SOLID
