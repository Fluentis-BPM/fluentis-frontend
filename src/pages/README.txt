# Directorio de Pages

Este directorio contiene los componentes principales de página que representan las diferentes vistas de la aplicación.

## Contenido
- Home.tsx: Página principal de la aplicación
- Login.tsx: Página de inicio de sesión
- Dashboard.tsx: Panel de control principal

## Propósito
Las páginas en este directorio:
- Representan las vistas principales de la aplicación
- Integran componentes y layouts
- Manejan la lógica específica de cada vista
- Implementan el enrutamiento de la aplicación

## Uso
Cada página debe:
- Utilizar un layout apropiado
- Mantener la lógica de presentación separada
- Implementar lazy loading cuando sea necesario
- Manejar estados y efectos de manera eficiente
