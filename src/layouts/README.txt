# Directorio de Layouts

Este directorio contiene los componentes de diseño que definen la estructura general de las páginas.

## Contenido
- MainLayout.tsx: Layout principal para usuarios autenticados
- AuthLayout.tsx: Layout para páginas de autenticación

## Propósito
Los layouts en este directorio:
- Proporcionan una estructura consistente en la aplicación
- Manejan la disposición común de elementos UI
- Implementan navegación compartida
- Gestionan estados globales de la interfaz

## Uso
Cada layout debe:
- Ser reutilizable entre múltiples páginas
- Mantener consistencia en el diseño
- Implementar responsive design
- Gestionar elementos comunes (header, footer, sidebar)
